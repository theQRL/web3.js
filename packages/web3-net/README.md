## Installation

You can install the package either using [NPM](https://www.npmjs.com/package/@theqrl/web3-net) or using [Yarn](https://yarnpkg.com/package/@theqrl/web3-net)

### Using NPM

```bash
npm install @theqrl/web3-net
```

### Using Yarn

```bash
yarn add @theqrl/web3-net
```
